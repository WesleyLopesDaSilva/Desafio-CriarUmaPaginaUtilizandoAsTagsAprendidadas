Tags Aprendidas no Primeiro Módulo

Voltar

No primeiro Módulo do curso de Formação HTML Web Developer, nós tivemos contatos com bastantes tags, desde tags para títulos a tags para ancoragem.

Segue abaixo algumas das tags vistas:

<h1> ao <h6> : Tags para títulos em ordem decrescente de tamanho. A tag <h1> é maior tag para títulos e a <h6> a menor.
<p> : Tag para criar um parágrafo para a sua página HTML
<blockquote> : Tag para indicar uma citação.
<mark> : Tag para destacar o texto.
<u> : Tag para sublihar.
<strong> : Deixar em Negrito.
<i> : Deixar em Itálico.
<small> : Para deixar um texto pequeno.
<a> : Tag para colocar link.
<ol> : Criar uma lista ordenada.
<ul> : Criar uma lista não ordenada
<li> : Criar os itens para as listas
<hr> : Traçar uma linha na horizontal, para separar os objetos.

Listas Ordenadas

Voltar

Lista Ordenada é o nome dado a uma lista criada em HTML, onde itens estão dispostos de forma ordenada. Os itens são precedidos por uma forma de contagem, podendo ser números, algarismos romanos, letras, entre outros.

Segue o modelo de como seria uma lista ordenada.

<ol>

<li> Primeiro item </li>
<li> Segundo item </li>
<li> Terceiro item </li>
</ol>

A saída do HTML escrito acima será:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Listas não Ordenadas

Voltar

Diferentemente da lista ordenada que é apresentada com uma certa numeração, indicando uma ordem, a lista não ordenada é apresentada apenas com marcadores, não indicando uma ordem.

<ul>

<li> Primeiro item </li>
<li> Segundo item </li>
<li> Terceiro item </li>
</ul>

A saída do HTML escrito acima será:

  • Primeiro item
  • Segundo item
  • Terceiro item
